Scheepsvaartmuseum re-design

Mini-project, 2019

This mini-project is based on increasing the conversionratio of a website. There where a few cases of museums and I chose, along with my team (two others) the Scheepsvaartmuseum.

The challenge was to make a design for a specific user group and test it with a programm which can exactly see where the user looks first, second, after that etc. For all the wireframes, download the PDF.
